## Plan: Busca eficiente para até 1000 produtos

Com até 1000 produtos em `produtos.json`, é viável otimizar tudo no frontend sem backend. O foco deve ser em carregamento único, indexação e busca rápida usando JavaScript.

### Steps
1. Carregar `produtos.json` via fetch ao iniciar a página (ex.: em `searchpage.js`).
2. Armazenar os produtos em um array/memória global para evitar múltiplos fetchs.
3. Implementar busca e filtros usando métodos eficientes (`Array.filter`, `Array.sort`).
4. Paginar resultados no frontend para evitar renderizar todos os produtos de uma vez.
5. (Opcional) Usar Web Worker para buscas/filtros se notar lentidão na interface.
